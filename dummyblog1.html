Loops in Javascript

Author:RamaChandran

What is a loop?

When you want to execute a block of code for "n" number of times then you use loops. There are mainly two types of loop Entry controlled loop and Exit Controlled loop.

Entry controlled Loop

When you check the condition at the beginning of the loop then it is called an entry controlled loop. For and while loop comes under Entry controlled loop.

Exit Controlled loop

When you check the condition at the beginning of the loop then it is called an exit controlled loop. Do while loop comes under Exit controlled loop.

For Loop

It is an entry controlled loop. Here initialization, test condition, and increment (or) decrement will be done on a single line. The loop will be executed until the condition is true.

Syntax

for(intialization;test condition;increment/decrement){
//block of code
}

while loop

It is an entry controlled loop. Here initialization, test condition, and increment (or) decrement should be done separately. The loop will be executed until the condition is true.

Syntax

while(condition)
{
//block of code
}

Eg:

do...while loop

It is an exit controlled loop. Here initialization, test condition, and increment (or) decrement will be done separately. The loop will be executed at least once even though the condition is true. It is mainly menu-based CLI app

Syntax

do
{
//block of code
}while(condition);
